# Arduino Face Detection System

Arduino-based project for face detection using ESP and camera modules, with a simple web server for interaction.

## Features
- Face detection using camera module
- Real-time processing
- Web interface (ESP Web Server)
- Communication between hardware and software components

## Technologies Used
- Arduino (C/C++)
- ESP (WiFi module / ESP32)
- Embedded systems programming
- Web server integration

## Project Structure
- `/arduino_cod` – Arduino source code
- `/ESP_WebServer` – web server implementation
- `project-documentation.pdf` – project documentation

## How It Works
1. The camera module captures images
2. The system processes data for face detection
3. Results are served through a web interface

## Notes
This project was developed as part of a practical assignment, focusing on embedded systems and hardware-software interaction.
